# Clock

### Create Word Clock

Create a clock thread to track time and light up specific LEDs to display the time.

```blocks	
lumaMatrix.createWordClock(
lumaMatrix.eMatrixVersion.V2,
0x007fff,
0xffff00,
0x00ff00
)
```
